# Mobazha Skills — Codex Installation

## Quick Start

```bash
git clone https://github.com/mobazha/mobazha-skills.git ~/.mobazha-skills
ln -sf ~/.mobazha-skills/skills ~/.codex/skills/mobazha-skills
```

## What It Does

Creates a symlink so Codex can discover Mobazha skills in its skills directory.

## Usage

After installation, ask Codex:

- "Help me deploy a self-hosted Mobazha store"
- "Install Mobazha native binary on my server"
- "Import my Shopify products into Mobazha"
